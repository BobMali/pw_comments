invertCommentSorting
""""""""""""""""""""
.. container:: table-row

   Property
      invertCommentSorting
   Data type
      boolean
   Default
      0
   Description
      Per default oldest comments are displayed first. If this option is true, the order of comments get inverted
      (newest comments first).
